Welcome to SIP Phone

SIP Phone is an WebRTC client which works using WebRTC & SIP protocol. You can use it in place of Soft (VoIP) Phone. It makes communication easy and cost-effective.

Key Features :
Call Type : Audio Call, Video Call
Call Assistant : Do not Disturb, Auto Answer Incoming Calls,
Call Features : Hold/Unhold, Mute/Unmute, Blind Transfer
